# Aula 3

## Tipos de dados importantes de serem lembrados:
- BLOB (Binary Large Object)
- TXT - Informações textuais bem grandes

## Alternativa para organização de dados em memória secundária:
- Arquivos Heap
- Arquivos Ordenado
- Arquivos Hashed

## Estrutura de Indexação:
- Bitmap index


## Tipos de índices:
- Índice primário (geralmente é mais eficiente que o índice secundário)
- Índice de agrupamento (clustering index)
- Índice secundário (auxiliar)
